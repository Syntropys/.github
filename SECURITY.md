# Security Policy

We appreciate reports of security vulnerabilities. Please **do not** open public issues for sensitive security problems.

## Reporting a Vulnerability
Use **GitHub Security Advisories** for this repository:
- Go to the **Security** tab
- Select **Advisories**
- Click **Report a vulnerability**

## Information to Include
Please provide:
- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Affected version/commit (if known)
- Suggested fix (optional)

## Supported Versions
We prioritize fixes for versions that are actively maintained.

## Response
We will respond to valid reports as quickly as possible and coordinate disclosure and fixes.
