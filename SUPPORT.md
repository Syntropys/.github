# Support

Thanks for using this project! We’re happy to help.

## How to Get Help
Please use the channels below:

1. **Issues (Bugs & Feature Requests)**  
   - Use the issue tracker for bugs or feature requests.  
   - Provide clear steps to reproduce and expected behavior.

2. **Discussions (Questions & Ideas)**  
   Use GitHub Discussions for general questions or ideas.

## Security Issues
Do **not** report security vulnerabilities in public issues.  
Please follow the instructions in **SECURITY.md**.

## Response Time
We respond to issues as time allows.
