# Contributing

Thank you for your interest in contributing! We welcome bug reports, feature requests, and pull requests.

## How to Contribute
1. Fork this repository.
2. Create a new branch (`feature/your-change`).
3. Make your changes with clear commits.
4. Open a Pull Request (PR) with a short description of what you changed.

## Reporting Bugs
- Use the issue tracker.
- Include steps to reproduce, expected behavior, and screenshots/logs if possible.

## Suggesting Features
- Explain the problem you want to solve.
- Provide a clear use-case or example.

## Code Style
- Keep code clean and readable.
- Follow existing project conventions.
- Add or update tests if applicable.

## Review Process
All PRs are reviewed by maintainers. Changes may be requested before merge.

## Code of Conduct
By participating, you agree to follow the Code of Conduct in this repository.
